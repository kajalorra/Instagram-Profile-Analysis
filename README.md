# 📊 Instagram Profile Analysis

Hi! 👋

This is my first Data Science project using Python. In this project, I analyzed an Instagram profile dataset and practiced the basic concepts of Exploratory Data Analysis (EDA).

While working on this project, I learned how to load data, understand a dataset, create charts, and find useful insights using Python.

---

## About the Dataset

The dataset contains information about 200 Instagram profiles.

The columns included are:

- Username
- Posts
- Followers
- Following
- Name
- Category
- Bio
- Link

---

## What I Did in This Project

In this project, I performed the following tasks:

- Imported the required Python libraries
- Loaded the dataset using Pandas
- Viewed the first and last few records
- Checked the shape of the dataset
- Explored column names and data types
- Generated summary statistics
- Checked for missing values
- Checked for duplicate records
- Counted unique values
- Analyzed profile categories
- Created charts to visualize the data
- Found the top 10 profiles based on followers
- Created a new column called **Follower Ratio**
- Saved the cleaned dataset

## How to Run

1. Download the project files.
2. Open Jupyter Notebook.
3. Open `Instagram_profile_analysis.ipynb`.
4. Run all the cells one by one.

If the required libraries are not installed, run:

```bash
pip install pandas numpy matplotlib
```
